# 数据分析样例说明

本目录包含了三个主要的数据分析样例，展示了如何对B站视频数据进行分析。每个样例都是一个独立的单元测试文件，可以直接运行查看结果。

## 环境准备

运行这些样例需要安装以下Python包：
```bash
pip install numpy pandas scikit-learn jieba mlxtend matplotlib
```

## 样例说明

### 1. 情感分析样例 (sentiment_analyzer_test.py)

展示了如何对视频评论进行情感分析，包括：
- TF-IDF特征提取
- 卡方检验特征选择
- 朴素贝叶斯分类
- 模型评估（准确率、召回率、F1分数）

运行方式：
```bash
python -m unittest tests/analyzer/sentiment_analyzer_test.py
```

### 2. 聚类分析样例 (clustering_analyzer_test.py)

展示了如何对视频内容进行主题聚类，包括：
- 文本向量化
- K-means聚类
- 轮廓系数评估
- 聚类结果分析和可视化

运行方式：
```bash
python -m unittest tests/analyzer/clustering_analyzer_test.py
```

### 3. 关联规则分析样例 (association_analyzer_test.py)

展示了如何分析用户观看行为中的关联模式，包括：
- 数据编码转换
- Apriori算法实现
- 关联规则挖掘
- 规则评估（支持度、置信度、提升度）

运行方式：
```bash
python -m unittest tests/analyzer/association_analyzer_test.py
```

## 使用建议

1. 每个测试文件都包含了详细的注释和打印输出，可以帮助理解代码的工作原理。

2. 测试用例中使用的数据都是模拟数据，在实际应用中需要替换为真实的B站数据。

3. 参数设置（如聚类数量、最小支持度等）可以根据实际需求调整。

4. 建议按照以下顺序学习：
   - 先运行情感分析样例，了解基本的文本处理和分类流程
   - 然后是聚类分析样例，学习如何发现数据中的模式
   - 最后是关联规则分析样例，了解如何挖掘用户行为模式

## 示例输出说明

每个测试用例都会输出详细的分析结果，例如：
- 情感分析会显示模型的评估指标
- 聚类分析会显示每个聚类的关键词和示例
- 关联规则分析会显示发现的规则及其评估指标

## 扩展建议

1. 可以尝试其他分类算法（如随机森林、SVM等）
2. 可以使用其他聚类算法（如DBSCAN、层次聚类等）
3. 可以调整特征提取方法（如Word2Vec、BERT等）
4. 可以添加数据可视化功能，更直观地展示分析结果

## 注意事项

1. 确保数据预处理步骤完整（去重、分词、去停用词等）
2. 注意参数调优对结果的影响
3. 在处理大规模数据时注意性能优化
4. 保存中间结果，便于调试和优化
